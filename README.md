# slide-stories

Slide stories is a project built using Vue.js 3 & Tailwind.css.

## Usage

- Just download or clone the repo.
- cd slide-stories
- run `npm install`
- run to serve

[demo]()

```
npm run serve
//or
yarn serve
```
