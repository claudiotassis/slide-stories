# slide-stories

Slide stories is a Simple project built using Vue.js 3 & tailwind.css

## Usage

- Just download or clone the repo.
- cd slide-stories
- run `npm install`
- run the **_json-server_** using `json-server db.json --watch --port 3000`
- run to serve

```
npm run serve
//or
yarn serve
```
